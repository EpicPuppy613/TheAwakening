The Awakening: A Custom Mindustry Experience

You wake up and find yourself in a strange place, it seems like you are in a deep chamber underground. In the dim light there is, you can barely make out alien looking glyphs on the walls. There is a ship nearby, you board it.
The controls on the ship seem so familiar, yet so vague. You start the ship up, and start exploring.

The Awakening is a custom Mindustry campaign made by EpicPuppy613.
This website is your guide, you will not be able to fully experience the campaign without it.

In case you don't have Mindustry, you can get it here from itch.io or here from Steam.
This campaign should function properly in Mindustry v6.x, and using other versions may break it.
There are 2 ways to experience this campaign, doing the full experience, where you have to set custom rules for yourself to do, or the basic version, which only contains the maps and win conditions.
